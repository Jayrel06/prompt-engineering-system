# Web Design Trends Research

Quick research on current web design trends for SaaS landing pages.

---

## Search Queries

- "web design trends 2025"
- "SaaS landing page trends 2025"
- "dark mode website design trends"
- "3D web design trends"
- "glassmorphism design trends"
- "hero section design trends"
- "website animation trends"

## Check These Sources

- awwwards.com (recent winners)
- godly.website (curated)
- dribbble.com (trending)
- Smashing Magazine articles

## Analyze

For each trend found:
- **Name**: What it's called
- **Stage**: Rising / Peak / Declining
- **Evidence**: Where seen (2+ sources)
- **Applicability**: How relevant to B2B SaaS
- **Implementation**: Easy / Medium / Hard

## Output

1. **Top 5 Trends to Adopt** (with evidence)
2. **Trends to Watch** (emerging)
3. **Trends to Skip** (declining or not relevant)

## Constraints

- Only include trends with multiple sources
- Note when trends are oversaturated
- Consider healthcare appropriateness
- Include performance implications
