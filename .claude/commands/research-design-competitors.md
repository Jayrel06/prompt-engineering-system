# Competitor Design Analysis

Analyze PT clinic software competitors vs premium SaaS benchmarks.

---

## Competitors to Analyze

**PT Clinic Software:**
- webpt.com
- cliniko.com
- simplepractice.com
- jane.app
- gethealthie.com

**Premium Benchmarks:**
- linear.app
- vercel.com
- stripe.com

## Analysis Dimensions

For each site, document:

### Visual
- Color scheme
- Typography
- Dark/light mode
- Animation usage
- Overall aesthetic (1-10)

### Technical
- Modern or dated?
- Mobile experience
- Performance feel

### Content
- Hero headline
- Value proposition clarity
- CTA copy
- Social proof approach

## Output

1. **Comparison Matrix** with scores
2. **Gap Analysis**: What competitors lack that benchmarks have
3. **Differentiation Opportunities**: How CoreReceptionAI can stand out
4. **Visual Elements to Implement**: Specific patterns from benchmarks

## Key Question

What do Linear, Vercel, and Stripe do that WebPT, Cliniko, etc. don't?
